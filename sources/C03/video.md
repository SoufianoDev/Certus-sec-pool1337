# 📹 C03 – Memory Layout, Structs & Data Structures (C Base)

## 🎯 Topic
C03 focuses on low-level memory understanding and data organization in C.

Core concepts:
- memory layout
- pointers
- struct pointers
- string manipulation
- memory traversal
- data structures basics

This stage introduces how complex data is represented and accessed in memory.
