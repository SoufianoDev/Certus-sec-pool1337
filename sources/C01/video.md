# 📹 C01 – Pointers & Memory (C Base)

## 🎯 Topic
C01 focuses on pointers and memory manipulation in C.

Core idea:
- variables in memory
- addresses (&)
- pointers (*)
- dereferencing
- modifying memory indirectly

This is the first step into real memory control in C.

## 🧠 Core Learning Focus

At this level, you should understand:

- `&` → get memory address
- `*` → access value at address
- pointer stores an address, not a value
- modifying memory indirectly through pointers
