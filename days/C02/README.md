# C02 - Strings Manipulation

## Description
This module focuses on handling strings in C and understanding how character arrays work.

## Concepts Covered
- String iteration
- Character arrays
- Basic string logic
- Manual implementation of string functions

## Objectives
- Understand how strings are stored in memory
- Learn to manipulate strings without libraries
- Strengthen pointer + array usage

## Notes
Strings in C are just arrays of characters ending with '\0'.
