# C00 - Basic C Concepts

## Description
This module introduces the fundamentals of C programming, focusing on basic syntax and output functions.

## Concepts Covered
- Writing simple C programs
- Using loops
- Using `putchar`
- Understanding compilation basics

## Objectives
- Build logical thinking in C
- Learn strict coding style (Norminette)
- Write clean and minimal code

## Key Constraints
- No forbidden functions
- Strict adherence to Norminette
- Simple and readable implementation

## Notes
This is the first step in mastering low-level programming fundamentals.
